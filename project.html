Predicting Crop Growth Using Weather Data¶

Introduction¶

Welcome to the Crop Growth Prediction project! Our objective is to create a smart tool using weather history and crop yield data to predict crop growth. By combining these datasets, we'll develop a machine learning model that forecasts crop performance based on past weather conditions.

Understanding how weather influences crops is crucial for smarter farming practices, offering farmers valuable insights to navigate unpredictable weather and optimize their yields.

def evaluate_model(model, model_name, X_train, y_train, X_val, y_val, X_test, y_test):
    # Fit the model
    model.fit(X_train, y_train)
    
    # Predict on validation and test sets
    y_val_pred = model.predict(X_val)
    y_test_pred = model.predict(X_test)
    
    # Calculate evaluation metrics for validation set
    mse_val = mean_squared_error(y_val, y_val_pred)
    r2_val = r2_score(y_val, y_val_pred)
    
    # Calculate evaluation metrics for test set
    mse_test = mean_squared_error(y_test, y_test_pred)
    r2_test = r2_score(y_test, y_test_pred)
    
    # Print evaluation metrics with model name
    print(f"Evaluation metrics for {model_name}:")
    print("Validation Set Metrics:")
    print(f"Mean Squared Error: {mse_val}")
    print(f"R-squared Score: {r2_val}\n")
    
    print("Test Set Metrics:")
    print(f"Mean Squared Error: {mse_test}")
    print(f"R-squared Score: {r2_test}")
